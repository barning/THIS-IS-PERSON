How To Play

The goal of the game is to show a desired object in the camera as quickly as possible. Every day, the object changes.

The algorithm examines each object seen in the camera and compares it to the requested object. Meanwhile, a timer is set.

If you hold the matching object in front of the camera and the algorithm is convinced, the game is over and you can brag to your friends about how quickly you finished it.

Credits

My name is Niklas Barning, UX designer and digital tinkerer from Bremen.

This game is written in JavaScript and uses the p5.js library for creating and manipulating the canvas element. The ml5.js library is used for object detection using the COCO-SSD model.
